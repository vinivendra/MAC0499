  gravity

Used to set the SCNScene’s Physics World’s gravity property. If the object is an NSNumber, the gravity will be a vector pointing in the negative y direction with the NSNumber as its magnitude. Otherwise, the object will be used to instantiate a Vector, which will then be set.

@property (nonatomic, strong) id gravity

Discussion

The getter always returns a Vector version of the physics world’s SCNVector3.
