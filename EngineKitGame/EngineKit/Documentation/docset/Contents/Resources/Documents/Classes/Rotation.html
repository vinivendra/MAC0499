Discussion

+ rotationWithDictionary:

Creates a Rotation based on the elements on a given NSDictionary. If there are elements with the key “x”, “y” and “z”(case insensitive), those elements are used for the axis; “a” or “w” is used for the angle. If one of them is missing, the corresponding “0”, “1” or “2” is used instead for the axis, and “3” for the angle. If any elements are still missing, the corresponding value is set to 0.0.
