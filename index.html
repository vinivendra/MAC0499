Welcome to EngineKit.

Though still under development, this project's goal is to wrap several frameworks available in iOS (ideally only those made by Apple itself) into one comprehensive system designed make game design as easy as it can be. It's not about reinventing the wheel and adding it to an already huge collection of wheels, it's about taking existing products and bringing them together.

To try it out, simply download a copy of the master branch. It usually contains the latest stable version, and should provide you with a project already setup and ready to start development.

To participate in development, the dev branch is your best bet. It's the project's main branch and where all the 'finished' features go before they are ready to be deployed to master. Just fork it, develop away, and create a pull request when your done.

Any doubts, suggestions and feedback (even negative feedback!) are welcome, whether it is about using the framework or developing it. You can probably reach me at vinivendra@gmail.com :)
