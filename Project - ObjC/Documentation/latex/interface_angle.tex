\subsection{Detailed Description}
Represents an \hyperlink{interface_angle}{Angle}, usually for a \hyperlink{interface_rotation}{Rotation}. Used mostly for initialization and conversion. More efficient when used exclusively with radians, but not slow unless used in a very tight loop.
