\subsubsection[{color\+With\+Array\+:}]{\setlength{\rightskip}{0pt plus 5cm}+ (Color $\ast$) color\+With\+Array\+: 
\begin{DoxyParamCaption}
\item[{(N\+S\+Array $\ast$)}]{array}
\end{DoxyParamCaption}
}\label{category_u_i_color_07_extension_08_acdc1ef989ee41b9b3ba7a97cd32bae83}
Creates a color with a given N\+S\+Array. The array must have from 1 to 4 N\+S\+Numbers.


\begin{DoxyItemize}
\item If there is 1 number only, it will be used as a white component (as in {\ttfamily -\/colorwith\+White}).
\item If there are 2 numbers, the first one will be used as a white component and the second one as an alpha value.
\item If there are 3 numbers, they will be used as R\+G\+B.
\item If here are 4 numbers, they will be used as R\+G\+B and alpha, in that order.
\end{DoxyItemize}

Initializes a U\+I\+Color based on the given object. The supported types are\+:


\begin{DoxyItemize}
\item U\+I\+Color, which creates a copy the color.
\item N\+S\+String, which works a lot like -\/color\+With\+Name.
\item N\+S\+Array, which works a lot like -\/color\+With\+Array.
\item N\+S\+Number, which works a lot like -\/color\+With\+White.
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
