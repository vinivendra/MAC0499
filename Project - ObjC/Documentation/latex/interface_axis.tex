\subsection{Detailed Description}
A representation of an axis (which is treated as an infinite vector and isn\textquotesingle{}t necessarily x, y or z). Used mainly for Rotations.

\item[{(N\+S\+String $\ast$)}]{string}

}\label{interface_axis_a9e8012b597263e5e31968621e8bd3a34}
Attempts to initialize an \hyperlink{interface_axis}{Axis} by scanning the string for numbers. If at least three numbers are found, the first three are used. Otherwise, scans the string for the characters x, y or z, and returns the corresponding axis.

\begin{DoxyWarning}{Warning}
If the \hyperlink{interface_axis}{Axis} fails to find something useful in the string, an {\ttfamily assert(false)} is triggered. 
\end{DoxyWarning}
