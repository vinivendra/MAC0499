\subsection{Detailed Description}
Represents a rotation of a determined angle around a determined axis.

Creates a \hyperlink{interface_rotation}{Rotation} based on the given N\+S\+Array. Trusts that the array has enough (4 or more) components, and that the first 4 components may be cast into {\ttfamily N\+S\+Numbers}.

The first 3 numbers are used to create an \hyperlink{interface_axis}{Axis}, and the fourth represents the angle for the rotation. This is much like the S\+C\+N\+Vector4s are used for rotation.
