- (NSArray *) map: (id(^)(id object))  block

Maps the given NSSet to NSArray. NOTE: If the block returns -nil, values are ignored. This is intentional to keep the behavior of -valueForKeyPath. This is different from NSArray -map, which puts NSNulls in the given case. If you want to preserve the count and use NSNulls, use set.allObjects -map:
